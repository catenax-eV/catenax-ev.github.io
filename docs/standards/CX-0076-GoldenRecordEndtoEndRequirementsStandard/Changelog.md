All notable changes to CX-0076 will be documented in this file.

## [1.6.0] - CX-Neptune Preview (26.06)

### Changed

- Extension of minimum requirements for data provisioning related to legal name and address data [see 2.1.2 UPLOAD CRITERIA AND MANDATORY FIELDS](./CX-0076-GoldenRecordEndtoEndRequirementsStandard.md#212-upload-criteria-and-mandatory-fields)
- Adjustments in logic for calculation of confidence level [see 2.1.12 CONFIDENCE LEVEL](./CX-0076-GoldenRecordEndtoEndRequirementsStandard.md#2112-confidence-level)
- Minor  adjustments related to standard template and updating technical references / links

## [1.5.0] - CX-Saturn (25.09)

### Added

- Added relations between legal entities [see 1.5.8 RELATIONS](./CX-0076-GoldenRecordEndtoEndRequirementsStandard.md#158-relations)

### Updated

- Adjusted wording for requirement regarding upload of BP data;  moved it from chapter [2.1.12 CONFIDENCE LEVEL](./CX-0076-GoldenRecordEndtoEndRequirementsStandard.md#2112-confidence-level) to chapter [see 2.1.2 UPLOAD CRITERIA AND MANDATORY FIELDS](./CX-0076-GoldenRecordEndtoEndRequirementsStandard.md#212-upload-criteria-and-mandatory-fields)
- Extended example for calculation of confidence level [see 2.1.12 CONFIDENCE LEVEL](./CX-0076-GoldenRecordEndtoEndRequirementsStandard.md#2112-confidence-level)
- Minor  adjustments related to standard template and updating technical references / links

## [1.4.0] - CX-Saturn (25.06)

### Added

- Added example for calculation of confidence level and adjusted wording to provide more precision [see 2.1.12 CONFIDENCE LEVEL](./CX-0076-GoldenRecordEndtoEndRequirementsStandard.md#2112-confidence-level)
- Minor adjustments related to standard template and updating technical references / links

### Updated

- Enhancement of chapter [see 2.1.8 QUALITY CHECKS FOR TAX NUMBER AND OTHER IDENTIFIER](./CX-0076-GoldenRecordEndtoEndRequirementsStandard.md#218-quality-checks-for-tax-number-and-other-identifier) with reference to standard CX-0010 where the identifier are thoroughly described

## [1.3.0] - CX-Io (24.12)

### Updated

- Adjustment of Table 2 in chapter [see 2.1.10 GOLDEN RECORD OUTPUT REQUIREMENTS](./CX-0076-GoldenRecordEndtoEndRequirementsStandard.md#2110-golden-record-output-requirements)
- Adjustment / enhancements of footnotes references regarding new standard versions
- Adapted to new template structure

### Removed

- Removed footnote for tax jurisdiction code as it is now technically implemented

## [1.2.0] - CX-Jupiter (24.08)

### Updated

- Adjustment of Table 2 in chapter [see 2.1.10 GOLDEN RECORD OUTPUT REQUIREMENTS](./CX-0076-GoldenRecordEndtoEndRequirementsStandard.md#2110-golden-record-output-requirements)
- Adjustment in chapter [2.1.12 CONFIDENCE LEVEL](./CX-0076-GoldenRecordEndtoEndRequirementsStandard.md#2112-confidence-level)
- Adjusted text for validation interval
- District and Region is now mandatory for Romania as listed in Table 2

### Added

- Added Tax Jurisdiction Code

## [1.1.0] - CX-Jupiter (24.06)

### Added

- New paragraph added at the end of chapter [2.1.2](./CX-0076-GoldenRecordEndtoEndRequirementsStandard.md#212-upload-criteria-and-mandatory-fields) providing more details on "own data"
- Chapter [2.1.11 NOTIFICATIONS](./CX-0076-GoldenRecordEndtoEndRequirementsStandard.md#2111-notifications) added
- Chapter [2.1.12 CONFIDENCE LEVEL](./CX-0076-GoldenRecordEndtoEndRequirementsStandard.md#2112-confidence-level) added
- Footnote for tax jurisdiction added in chapter [2.1.8](./CX-0076-GoldenRecordEndtoEndRequirementsStandard.md#218-quality-checks-for-tax-number-and-other-identifier)

### Updated

- Adjustment of Table 2 in chapter [2.1.10 GOLDEN RECORD OUTPUT REQUIREMENTS](./CX-0076-GoldenRecordEndtoEndRequirementsStandard.md#2110-golden-record-output-requirements)
- Street is now neither mandatory nor optional for Japan as listed in Table 2
- Chapter [2.1.6](./CX-0076-GoldenRecordEndtoEndRequirementsStandard.md#216-quality-checks-for-legal-entity-including-legal-form) has been expanded specifying expected relations and quality checks in more detail

### Removed

- Data field references deleted.
