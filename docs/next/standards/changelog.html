A) Updated Standards​

Note: Detailed information regarding the changes can be found in the standard documents under the section “Comparison with the Previous Version of the Standard”.

Network Services​

CX-Nr.Standard NameVersionManagement Summary
CX-0018Dataspace Connectivity4.1.0- Specified default credential context for catalog requests
- Updated to final versions of DSP 2025-1 and DCP 1.0
- Finalized discovery, protocol handling, and backward compatibility
- Policy constraints now covered in a separate standard
CX-0001EDC Discovery API1.2.0- Deprecated EDC Discovery API (moved to backward compatibility section)
- Changed purpose: now specifies Participant Agent registration
- Title and terminology updated to "Participant Agent"
CX-0049DID Document2.2.0- Added JSON schema for validation
- Added DataService and CredentialService to service array
- Rewrote introduction and removed CatalogService field
CX-0149Verified Company Identity2.0.0- Added requirements for wallets and guidance on obtaining them
- Scope now covers wallet requirements and acquisition
- Credential descriptions moved to CX-0050; OIDC references removed
CX-0050Framework Agreement Credential2.2.0- Scope expanded to all Catena-X credentials
- Added membership, BPN, and dismantler credentials
- All credential types now include usage description and JSON schema
- Introduction rewritten

CX-0074Business Partner Gate API4.1.0- Added relations entity and definitions for managing/legal entities

B) New Standards​

CX-Nr.Standard NameVersionManagement Summary
CX-0154Digital Master Data1.0.0- Standardized, interoperable, and secure exchange of structured master data (product, material, 2D & 3D data)
- Reduces inconsistencies and manual effort
- Improves efficiency in PDM/CAD systems
- Accelerates digitalization and collaboration
- Supports compliance and new digital services

- Establishes unified data model for requirements
- Based on existing models and standards
CX-0153Tariffs Use Case1.0.0- Standardizes tariff data collection and reporting
- Replaces manual processes with secure, automated solutions
- Sets standards for data aggregation, reporting, and quality controls
- Focus on raw material categories and mapping “Country of Origin” codes
- Lays groundwork for long-term system integration

- Eliminates duplication and inconsistencies
- Ensures all use cases reference this standard via CX-0018
- Reduces maintenance effort and risk of errors
- No dependencies on other standards
